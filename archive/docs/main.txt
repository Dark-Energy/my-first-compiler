
        Компилятор состоит из лексическго анализатора, синтаксического
анализатора, построителя деревеьев, генератора кода и таблицы символов.
Компиляция выполняется за два прохода: сначала производится 
синтаксический анализ текста программы и созданиее промежуточного 
представления в виде дерева, затем оптимизация и генерация объектного кода.
        Синтаксический анализатор вызывает лексический анализатор для 
получения следующей лексемы. При распознании каждого правила, анализатор
вызывает построитель дерева, для создания очередного узла.
        Часть оптимизаций выполняется на стадии разбора программы и 
генерации дерева. Это:
        1) вычисление константных выражений
        2) алгебраические упрощения
        3) оптимизация логических выражений (при вычислении логических
выражениее по короткой схеме)
        4) частичное удаление неиспользуемого кода (код распознается, но
исключается из промежуточного представления, из дерева)
        При создании узлов дерева, также производится проверка совместимости
типов в выражениях. 


ГЕНЕРАЦИЯ КОДА
        Генератор кода получает на входе дерево разбора, построенное 
анализатором. Входные условия следующие:
        1) Дерево программы не содержит ошибок
        2) проведены оптимизации, описанные выше
